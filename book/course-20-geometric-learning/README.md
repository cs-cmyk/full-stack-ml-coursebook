# Course 20: Geometric and Structured Learning

Graph neural networks and geometric deep learning enable AI systems to reason about relationships and structured data. Advanced GNN architectures, real-world graph applications from drug discovery to fraud detection, and methods for learning from non-Euclidean domains like point clouds and manifolds.

## Prerequisites

- Courses 1-11 or equivalent knowledge in
- Machine Learning fundamentals (Course 4)
- Deep Learning (Course 5)
- Graph Data Science basics (Course 10, Module 27)

## Chapters

| Chapter | Title | Key Topics |
|---------|-------|------------|
| ch57 | Advanced Graph Neural Networks | Message passing framework, GCN/GraphSAGE/GAT/GIN architectures, Graph Transformers, heterogeneous graphs, temporal graphs, scalable GNNs |
| ch58 | Applications of Graph ML | Molecular property prediction, drug discovery, social network analysis, fraud detection, traffic optimization, knowledge graph embeddings, graph-based recommendations |
| ch59 | Other Structured Domains | Point cloud processing (PointNet/PointNet++), manifold learning, equivariant neural networks, set/sequence-to-set models, combinatorial optimization with ML |
