# Chapter 27 Diagrams: Classical NLP

This directory contains all diagrams for Chapter 27 on Classical NLP.

## Generated Diagrams

The following diagrams are generated by the `generate_all_diagrams.py` script:

1. **preprocessing_pipeline.png** - Visual flowchart showing the text preprocessing pipeline from raw text through tokenization, normalization, and stemming/lemmatization
2. **bow_matrix_heatmap.png** - Heatmap visualization of the Bag-of-Words document-term matrix showing word counts across documents
3. **bow_vs_tfidf.png** - Side-by-side comparison of raw BoW counts vs TF-IDF weighted values for a sample document
4. **ngram_extraction.png** - Illustration of n-gram extraction showing unigrams, bigrams, and trigrams from a sentence, highlighting how bigrams capture negation patterns
5. **similarity_comparison.png** - Heatmap comparison of BoW vs TF-IDF cosine similarity matrices on 20newsgroups dataset

## Usage

To regenerate all diagrams:

```bash
python generate_all_diagrams.py
```

The script will:
- Generate all 5 diagrams with consistent styling
- Use the specified color palette (blue, green, orange, red, purple, gray)
- Ensure all figures are 150 DPI
- Keep width at or below 800px
- Include progress reporting and error handling

## Requirements

- matplotlib
- numpy
- pandas
- seaborn
- scikit-learn
- Pillow (for verification)
