# Course 1: Foundations of Data Science

This course builds the mathematical foundation required for data science. Master linear algebra, calculus, probability, and statistics — the core tools behind machine learning algorithms and statistical analysis.

## Prerequisites

None.

## Chapters

| Chapter | Title | Key Topics |
|---------|-------|------------|
| ch01 | Linear Algebra | Vectors, matrices, eigenvalues/eigenvectors, SVD, linear transformations |
| ch02 | Calculus | Derivatives, gradients, chain rule, optimization, multivariable calculus |
| ch03 | Probability | Sample spaces, Bayes' theorem, distributions, CLT, law of large numbers |
| ch04 | Statistics | Descriptive stats, hypothesis testing, confidence intervals, p-values, Bayesian vs frequentist |

## How to Use

Each chapter is available as:
- **content.md** — Read as markdown
- **content.ipynb** — Open as Jupyter notebook (runnable code + visualizations)

```bash
cd book/course-01-foundations/
jupyter lab
```
