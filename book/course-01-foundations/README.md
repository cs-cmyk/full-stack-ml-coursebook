# Course 01: Mathematical Foundations

Build the mathematical foundation required for data science. Master linear algebra, calculus, probability, and statistics — the core tools behind machine learning algorithms and statistical analysis.

## Prerequisites

None.

## Chapters

| Chapter | Title | Key Topics |
|---------|-------|------------|
| [1](ch01-linear-algebra/) | Linear Algebra for Data Science | Vectors, matrices, eigenvalues/eigenvectors, SVD, linear transformations |
| [2](ch02-calculus/) | Calculus for Data Science | Derivatives, gradients, chain rule, optimization, multivariable calculus, cross-entropy derivation |
| [3](ch03-probability/) | Probability for Data Science | Sample spaces, Bayes' theorem, distributions, CLT, law of large numbers |
| [4](ch04-statistics/) | Statistics for Data Science | Descriptive stats, hypothesis testing, confidence intervals, p-values, Bayesian vs frequentist |

## How to Use

Each chapter is available as:
- **content.md** — Read as markdown
- **content.ipynb** — Open as Jupyter notebook (runnable code + visualizations)

```bash
cd book/course-01-foundations/
jupyter lab
```
