# Chapter 3: Probability - Diagrams

This directory contains all visualizations for the Probability chapter.

## Generated Diagrams

### 1. probability_concepts.png (207 KB)
**4-panel educational visualization showing core probability concepts:**
- Panel A: Sample Space with Venn diagram (events A, B, their intersection and union)
- Panel B: Conditional Probability tree diagram showing P(A|B)
- Panel C: Bayes' Theorem flow diagram (Prior → Evidence → Posterior)
- Panel D: Common Distributions (Bernoulli, Binomial, Uniform, Normal)

**Referenced in:** Visual section (line 56)

### 2. law_of_large_numbers.png (142 KB)
**Demonstrates convergence of empirical probabilities:**
- Left panel: Fair coin (P(H) = 0.5) convergence over 1,000 flips
- Right panel: Unfair coin (P(H) = 0.7) convergence over 1,000 flips
- Shows annotations at key flip counts (10, 100, 1,000)

**Generated by:** Code Example Part 1 (line 138)

### 3. distribution_fitting.png (154 KB)
**Shows how probability distributions model real data:**
- Left panel: Iris sepal length histogram with fitted Normal distribution
- Right panel: Actual vs synthetic data comparison
- Includes mean, ±1σ markers, and probability calculations

**Generated by:** Code Example Part 3 (line 308)

### 4. expected_value_variance.png (142 KB)
**Visualizes expected value and variance in house prices:**
- Left panel: California housing prices with E[X] as balance point
- Right panel: Price distributions by income region (Low/Medium/High)
- Shows how higher variance = more uncertainty

**Generated by:** Code Example Part 4 (line 410)

## Color Palette (Consistent Across All Diagrams)
- Blue (#2196F3): Primary data/concepts
- Green (#4CAF50): Secondary data/success
- Orange (#FF9800): Highlights/warnings
- Red (#F44336): Important markers/errors
- Purple (#9C27B0): Prior/hypothesis
- Gray (#607D8B): Supplementary info
