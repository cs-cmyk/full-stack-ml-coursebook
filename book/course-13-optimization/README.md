# Course 13: Advanced Optimization

Optimization theory and advanced training techniques for deep learning — from convex optimization to Bayesian optimization and AutoML.

## Prerequisites

Courses 1 (Foundations) and 5 (Deep Learning).

## Chapters

| Chapter | Title | Key Topics |
|---------|-------|------------|
| [36](ch36-optimization-theory/) | Optimization Theory | Convex optimization, KKT conditions, second-order methods, constrained optimization |
| [37](ch37-advanced-training/) | Advanced Training Techniques | Learning rate schedules, mixed precision, distributed training, gradient accumulation |
| [38](ch38-bayesian-optimization/) | Bayesian Optimization | Acquisition functions, surrogate models, NAS, automated model selection |

## How to Use

Each chapter is available as:
- **content.md** — Read as markdown
- **content.ipynb** — Open as Jupyter notebook (runnable code + visualizations)

```bash
cd book/course-13-optimization/
jupyter lab
```
