## Color Palette

All diagrams use a consistent color scheme:
- **Blue** (#2196F3): Data/inputs
- **Green** (#4CAF50): Success states/efficient methods
- **Orange** (#FF9800): Intermediate processes
- **Red** (#F44336): Inefficient/complex methods
- **Purple** (#9C27B0): Advanced techniques
- **Gray** (#607D8B): Supporting elements

## Regenerating Diagrams

To regenerate the matplotlib diagrams:
```bash
python generate_lora_architecture.py
python generate_rlhf_vs_dpo.py
```

Both scripts save diagrams at 150 DPI with white backgrounds for print quality.
