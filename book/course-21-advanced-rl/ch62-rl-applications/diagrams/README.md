## Color Palette

All diagrams use a consistent color scheme:
- Blue (#2196F3): Primary elements, components
- Green (#4CAF50): Positive outcomes, RL solutions
- Orange (#FF9800): Highlights, starting points
- Red (#F44336): Connections, real-world distributions
- Purple (#9C27B0): Special elements (e.g., KL penalty)
- Gray (#607D8B): Secondary elements, grids
