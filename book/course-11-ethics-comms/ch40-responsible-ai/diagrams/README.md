# Chapter 40: Responsible AI - Diagrams

This directory contains all visualizations for Chapter 40: Responsible AI.

## Generated Diagrams

### 1. bias_pipeline.png
**Type:** Flowchart (matplotlib)
**Purpose:** Illustrates the ML pipeline stages and where different types of bias enter
**Features:**
- Shows 5 pipeline stages: Data Collection → Feature Engineering → Model Training → Model Deployment → Decision Making
- Annotates 4 types of bias with color-coded arrows:
  - Historical Bias (red)
  - Measurement Bias (orange)
  - Aggregation Bias (purple)
  - Deployment Bias (gray)
- Dimensions: 12x6 inches, 150 DPI

### 2. adult_income_bias.png
**Type:** Bar charts (matplotlib/seaborn)
**Purpose:** Shows income disparities in the Adult Income dataset by gender and race
**Features:**
- Left panel: Gender disparity (Female vs Male) with percentages
- Right panel: Race disparity (horizontal bars) sorted by income rate
- Color scheme: Orange/Blue for gender, Green for race
- Clear annotations showing exact percentages
- Dimensions: 14x5 inches, 150 DPI

### 3. fairness_tradeoff.png
**Type:** Grouped bar chart (matplotlib)
**Purpose:** Compares model performance before and after debiasing
**Features:**
- Shows 3 metrics: Accuracy, Demographic Parity Difference, TPR Difference
- Red bars: Baseline model
- Green bars: After debiasing
- Demonstrates the accuracy-fairness trade-off
- Value labels on each bar
- Dimensions: 10x6 inches, 150 DPI

### 4. differential_privacy.png
**Type:** Multi-panel histogram (matplotlib)
**Purpose:** Visualizes privacy-utility trade-off with different epsilon values
**Features:**
- 3 panels showing ε = 0.1 (Strong), 1.0 (Moderate), 10.0 (Weak)
- Histograms of 100 noisy query results
- Red dashed line shows true mean salary
- Demonstrates how noise spreads decreases with higher epsilon
- Dimensions: 15x5 inches, 150 DPI

### 5. dp_count_tradeoff.png
**Type:** Multi-panel histogram (matplotlib)
**Purpose:** Shows differential privacy applied to counting queries
**Features:**
- 3 panels with ε = 0.1, 1.0, 10.0
- Histograms showing distribution of noisy counts
- Red dashed line marks true count
- Illustrates privacy-accuracy trade-off for counting
- Dimensions: 15x5 inches, 150 DPI

## Color Palette

- Red: #F44336 (error/alert)
