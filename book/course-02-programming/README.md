# Programming for Data Science

Essential programming skills for data science, covering Python, SQL, visualization, and development tools. Manipulate data, create visualizations, query databases, and use version control for collaborative work.

## Prerequisites

- Course 1 (Foundations of Data Science)

## Chapters

| Chapter | Title | Key Topics |
|---------|-------|------------|
| ch05 | Python Fundamentals | Data types, control flow, functions, OOP, error handling, file I/O |
| ch06 | Python for Data Science | NumPy arrays, Pandas DataFrames, data wrangling, datetime handling |
| ch07 | Data Visualization | Matplotlib, Seaborn, Plotly, visualization best practices |
| ch08 | SQL & Databases | SELECT, JOINs, aggregation, subqueries, window functions, database design |
| ch09 | Version Control | Git, GitHub, branching, collaboration workflows |

## How to Use

Each chapter is available as:
- **content.md** — Read as markdown
- **content.ipynb** — Open as Jupyter notebook (runnable code + visualizations)

```bash
cd book/course-02-programming/
jupyter lab
```
