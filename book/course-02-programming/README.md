# Course 02: Programming for Data Science

Master Python and essential data science tools — from core language features through NumPy, Pandas, visualization, SQL, and version control.

## Prerequisites

Basic programming experience helpful but not required.

## Chapters

| Chapter | Title | Key Topics |
|---------|-------|------------|
| [5](ch05-python-fundamentals/) | Python Fundamentals for Data Science | Data structures, functions, OOP, best practices |
| [6](ch06-python-data-science/) | Python for Data Science: NumPy and Pandas | Arrays, broadcasting, DataFrames, groupby, merging, reshaping |
| [7](ch07-data-visualization/) | Data Visualization with Matplotlib and Seaborn | Matplotlib, Seaborn, Plotly, storytelling with data |
| [8](ch08-sql-databases/) | SQL Databases for Data Science | SQL queries, joins, aggregations, database design |
| [9](ch09-version-control/) | Version Control with Git | Git fundamentals, branching, collaboration, CI/CD basics |

## How to Use

Each chapter is available as:
- **content.md** — Read as markdown
- **content.ipynb** — Open as Jupyter notebook (runnable code + visualizations)

```bash
cd book/course-02-programming/
jupyter lab
```
