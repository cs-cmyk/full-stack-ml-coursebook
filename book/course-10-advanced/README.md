# Course 10: Advanced & Specialized Topics

Master specialized domains in data science including recommendation systems, reinforcement learning, computer vision, graph neural networks, and causal inference. Cutting-edge techniques powering real-world applications from personalized recommendations to autonomous systems.

## Prerequisites

Courses 4 (Machine Learning) and 5 (Deep Learning), or knowledge of supervised/unsupervised learning, neural networks, and deep learning frameworks (PyTorch/TensorFlow).

## Chapters

| Chapter | Title | Key Topics |
|---------|-------|------------|
| ch35 | Recommendation Systems | Collaborative filtering, matrix factorization, hybrid approaches, deep learning for recommendations, evaluation metrics (precision@k, NDCG) |
| ch36 | Reinforcement Learning | Markov Decision Processes, Q-learning, DQN, policy gradients, actor-critic methods, applications (robotics, RLHF) |
| ch37 | Computer Vision | Image preprocessing, object detection (YOLO, Faster R-CNN), segmentation (U-Net, Mask R-CNN), pose estimation, OCR |
| ch38 | Graph Data Science | Graph theory, network analysis, Graph Neural Networks (GCN, GraphSAGE, GAT), knowledge graphs |
| ch39 | Causal Inference | A/B testing, causal graphs, propensity score matching, difference-in-differences, instrumental variables, uplift modeling |

## How to Use

Each chapter is available as:
- **content.md** — Read as markdown
- **content.ipynb** — Open as Jupyter notebook (runnable code + visualizations)

cd book/course-10-advanced/
