# Course 10: Advanced Topics Survey

Survey of advanced ML domains — recommender systems, reinforcement learning, computer vision, graph ML, and causal inference. Each chapter provides foundations before the deep-dive courses in Part V.

## Prerequisites

Courses 4 (Machine Learning) and 5 (Deep Learning).

## Chapters

| Chapter | Title | Key Topics |
|---------|-------|------------|
| [35](ch35-recsys/) | Recommender Systems | Collaborative filtering, content-based, matrix factorization, deep recommendations |
| [36](ch36-rl/) | Reinforcement Learning Foundations | MDPs, value functions, Q-learning, policy gradients |
| [37](ch37-cv/) | Computer Vision Foundations | Image preprocessing, augmentation, object detection basics |
| [38](ch38-graphs/) | Graph Theory Fundamentals | Graph representations, traversals, centrality, community detection |
| [39](ch39-causal/) | Causal Inference & Experimentation | A/B testing, causal graphs, confounders, natural experiments |

## How to Use

Each chapter is available as:
- **content.md** — Read as markdown
- **content.ipynb** — Open as Jupyter notebook (runnable code + visualizations)

```bash
cd book/course-10-advanced/
jupyter lab
```
