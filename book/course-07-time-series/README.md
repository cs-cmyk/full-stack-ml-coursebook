# Course 7: Time Series Analysis & Forecasting

Time series analysis equips you with the tools to forecast future values from sequential, time-ordered data—essential for predicting sales, stock prices, energy demand, and more. Classical statistical methods (ARIMA, SARIMA) and modern techniques (Prophet, tree-based models, LSTMs) for understanding temporal patterns and building production-ready forecasting systems.

## Prerequisites

- **Course 3: Exploratory Data Analysis & Feature Engineering** — Understanding data patterns and feature engineering
- **Course 4: Machine Learning** — Supervised learning, model evaluation, and validation techniques

Familiarity with Python (NumPy, Pandas) and basic statistics (mean, variance, correlation).

## Chapters

| Chapter | Title | Key Topics |
|---------|-------|------------|
| ch29-classical-ts | Classical Time Series Analysis | Decomposition (trend, seasonality, noise), stationarity, ACF/PACF, ARIMA, SARIMA, exponential smoothing, Holt-Winters, STL decomposition |
| ch30-advanced-ts | Advanced Time Series Analysis | Prophet, tree-based forecasting (XGBoost), LSTMs, multivariate time series, walk-forward validation, evaluation metrics (MAE, MAPE, SMAPE) |
