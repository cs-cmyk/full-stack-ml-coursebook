# Course 07: Time Series & Forecasting

Classical and deep learning approaches to time series analysis, forecasting, and anomaly detection.

## Prerequisites

Courses 1 (Foundations) and 4 (Machine Learning).

## Chapters

| Chapter | Title | Key Topics |
|---------|-------|------------|
| [29](ch29-classical-ts/) | Classical Time Series Analysis | Stationarity, decomposition, ARIMA, Prophet, seasonality |
| [30](ch30-advanced-ts/) | Advanced Time Series Analysis | Deep forecasting, anomaly detection, walk-forward validation |

Each chapter is available as:
- **content.md** — Read as markdown
- **content.ipynb** — Open as Jupyter notebook (runnable code + visualizations)

cd book/course-07-time-series/
